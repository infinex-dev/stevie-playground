# stevie-playground

A small TypeScript library of formatting utilities. Used as a smoke fixture for the
[Stevie](https://github.com/infinex-xyz/stevie) agent framework.

## Canary Workload

This repository serves as a **canary workload** for the Stevie agent framework. It provides a controlled environment to validate that agents can correctly read, modify, test, and commit changes to a TypeScript codebase. By running agent tasks against this playground, the team can detect regressions or issues in the agent framework before deploying to production repositories.

## Modules

- `slugify` — convert a string to a URL-safe slug.
- `currency` — format and parse currency amounts (`USD 12.34`).
- `time` — format ISO timestamps and produce relative-time strings.
- `bytes` — format and parse byte counts (`1.5 MB`).
